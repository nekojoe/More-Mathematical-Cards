# More Mathematical Cards
    more 'maths themed' cards for D&DG

# Additions
    this mod currently adds 28 new cards to the game:
    - e of each suit
    - +- of each suit
    - set of each suit
    - approximation of each suits
    - delta of each suit
    - ratio of each suit
    - sine of each suit
